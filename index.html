VieStatic

The simplest way to add a floating QR badge to any website — with one line of JavaScript.

Live Demo

This page is running VieStatic right now.

Look at the corner — click the floating badge to open the QR modal.

Install

Drop this script anywhere in your HTML. That’s it.

<link rel="stylesheet" href="viestatic.css">
<script src="viestatic.js" data-pos="3"></script>

Positions:

1 = top-left
2 = bottom-left
3 = top-right
4 = bottom-right

Custom URL

By default, VieStatic uses the current page URL. You can override it:

<script src="viestatic.js" data-pos="4" data-url="https://example.com"></script>

JavaScript API

You can also initialize VieStatic manually:

<script>
  VieStatic.init({
    url: "https://example.com",
    position: 1
  });
</script>
